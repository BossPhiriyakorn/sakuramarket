# 🚀 Quick Start

```bash
npm install
npm run dev
```

เปิด: `http://localhost:3000`

## 🎮 Controls

- **Pan:** Click + Drag
- **Zoom:** Mouse Wheel
- **Select:** Click Parcel

## 📁 Files

- `app/cms/data/cmsData.ts` - ข้อมูล CMS (ว่างจนเชื่อม API)
- `constants.ts` - Config
- `components/` - PixiJS components

---

**แอปเดียว | Frontend Only | No Backend**
